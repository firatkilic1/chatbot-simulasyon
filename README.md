# Dr.Coder Chatbot

Basit ve eğlenceli bir HTML, CSS ve JavaScript tabanlı chatbot arayüzü. Bu proje, kullanıcı ile yapay zekâ karakteri **Dr.Coder** arasında geçen yazışmaları simüle eder.

## Proje Dosya Yapısı

📂 drcoder-chatbot  
├── index.html → Ana HTML dosyası  
├── style.css → Chat UI stilleri  
└── script.js → Mesaj simülasyon mantığı

## Özellikler

- Kullanıcı ve bot arasında sırayla yazışma
- Yazı yazma animasyonu (typewriter effect)
- Botun yazıyor efekti (3 noktalı typing animasyonu)
- Responsive, sade ve modern arayüz

## Nasıl Çalıştırılır?

1. Proje klasörünü VSCode'da aç.
2. `index.html` dosyasını bir tarayıcıda çalıştır.
3. Sohbet otomatik olarak başlar.

## Kullanılan Teknolojiler

- HTML5
- CSS3 (Flexbox, Animasyonlar)
- JavaScript (DOM, zamanlayıcılar)
